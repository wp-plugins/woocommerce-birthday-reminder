Send an email to the users with birthday in the current month and to the users with birthday in the current day.

== Description ==

This plugin sends an email to users with birthday in current month and for users with birthday in present day. It also provides an administrative panel under Woocommerce Emails where the administrator can customize the contents of email, send which one among other things.

= Descrição em Português =

Este plugin envia um e-mail para o usuário aniversariante do mês e/ou para o usuário aniversariante do dia, também provê um painel administrativo onde o administrador é capaz de personalizar o conteúdo do e-mail, qual mandar, entre outras funcionalidades.

== Installation ==

1. Upload the contents of `woocommerce-birthday-reminder` to the `/wp-content/plugins/woocommerce-birthday-reminder/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I have to install WooCommerce and WooCommerce Extra Checkout Fields for Brazil to use this plugin? =

Yes. The plugin was been tested under woocommerce 2.2.8 and WooCommerce Extra Checkout Fields for Brazil 3.3.1, so that might be enough.
